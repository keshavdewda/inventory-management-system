# Inventory Management System

A full-stack inventory management system designed to manage products, orders, and business operations efficiently with role-based access control.

## 🚀 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MongoDB

## ✨ Features
- Interactive dashboard with analytics
- Product and inventory management
- Order tracking and management
- Role-based access (Admin, Manager, User)
