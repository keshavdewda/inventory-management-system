# 📦 Inventory Management System

A full-stack Inventory Management System built using React, Node.js, and MongoDB. It helps manage products, orders, suppliers, and users with role-based access.

## 🔗 Live Demo

## 👥 Role-Based Access

* Admin – Full control (products, users, reports)
* Manager – Manage operations (orders, suppliers)
* Viewer – Read-only access

## 🧰 Tech Stack

Frontend: React (Vite), Axios, React Router
Backend: Node.js, Express.js, JWT
Database: MongoDB Atlas
Deployment: Vercel & Render

## ✨ Features

* Secure authentication using JWT
* Role-based authorization
* Product management (CRUD)
* Order tracking with stock updates
* Supplier management
* Dashboard with charts and KPIs
* Sales & inventory reports
* Responsive dark UI

## ⚙️ Run Locally

### Clone repository

git clone https://github.com/your-username/inventory-management-system.git
cd inventory-management-system

### Backend setup

cd inventory-backend
npm install

Create .env file:
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
FRONTEND_URL=http://localhost:5173

Run backend:
npm run dev

### Frontend setup

cd inventory-management-app
npm install

Create .env file:
VITE_API_URL=http://localhost:5000

Run frontend:
npm run dev
